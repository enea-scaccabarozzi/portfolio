# Security Policy

## Supported Versions

Only the latest release is supported with security updates.

## Reporting a Vulnerability

**Please do NOT open public issues for security vulnerabilities.**

Use [GitHub Security Advisories](https://github.com/enea-scaccabarozzi/portfolio/security/advisories/new) to report vulnerabilities privately.

You can expect an initial response within 7 business days.

## Bug Bounty

This project does not have a bug bounty program.
