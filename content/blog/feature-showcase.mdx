### TypeScript

```typescript
interface BlogPost {
  title: string
  slug: string
  date: string
  tags: string[]
}

function getPostBySlug(posts: BlogPost[], slug: string): BlogPost | undefined {
  return posts.find(post => post.slug === slug)
}

const featured = getPostBySlug(posts, "feature-showcase")
console.log(featured?.title)
```

### Python

```python
from dataclasses import dataclass
from datetime import date

@dataclass
class BlogPost:
    title: str
    slug: str
    published: date
    tags: list[str]

    @property
    def permalink(self) -> str:
        return f"/blog/{self.slug}"

posts = [
    BlogPost("Demo", "demo", date.today(), ["test"]),
]
```

## Alerts

Custom callout components with four semantic variants.

  This is an informational callout. Use it for supplementary context or tips that enhance understanding.
</Alert>

<Alert variant="warning" title="Warning">

## Lists

### Unordered List

- Static site generation with Next.js
- MDX content pipeline via Velite
- Syntax highlighting with shiki
- Type-safe content schemas

### Ordered List

1. Write content in MDX
2. Velite compiles at build time
3. rehype plugins process the AST
4. Static HTML is generated

### Nested List

- Content pipeline
  - Velite processes `.mdx` files
  - Frontmatter is validated against a Zod schema
  - The output is type-safe TypeScript
- Rendering
  - MDX compiles to a function body
  - Custom components are injected at render time
  - Prose styling is handled by `@tailwindcss/typography`
